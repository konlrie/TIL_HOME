### 1.2 기본 사용법

* `requests.get(URL)` : Response 객체 리턴
* `requests.get(URL).text` : Response 내용이 문자열(str)로 리턴
* `requests.get(URL).status_code` : 상태 코드 리턴(200, 404, ...)

## 3. Python 문법 +

### 3.1 문자열 삽입 (문자열 포매팅)

* 파이썬 3.0 이상 -> **format 함수를 이용한 포매팅**

* 파이썬 3.6 이상 -> **f-string 포매팅**

```python
# format 함수
'{1} {0}'.format('one', 'two')

# f-string
a, b = 'one', 'two'
f'{a}, {b}'
```
